About Holochain

Bitcoin’s central mechanism — the Blockchain — is a monumental achievement in computer science. And from that central achievement, many other cryptocurrencies have emerged attempting to improve the model in one way or another. Holochain has come along to further decentralize, maximize efficiency, and allow for all types of interfaces and applications to be built with it. Therefore, Holochain is a data integrity engine for distributed peer-to-peer applications.
